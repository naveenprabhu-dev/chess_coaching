# Chess Coaching - Personal Site
A simple website for my chess coaching services: who I am, what I offer, and a contact form to reach me. 

# Tech Stack
Framework: Next.js
UI: React, Tailwind CSS
Hosting: Vercel

# Modifying
1) Clone
git clone https://github.com/YOUR-USER/YOUR-REPO.git
cd YOUR-REPO

2) Install deps
npm install

3) Run dev server
npm run dev
